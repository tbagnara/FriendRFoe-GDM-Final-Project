# FriendRFoe-GDM-Final-Project
Multiplayer platformer meyhem
### How to test multiplayer
(To play this game you must have the Unity Editor installed and open the project through it)
1. Create a client clone of the project using parrelsync
2. Open both projects at the "Players" scene
3. The game will automatically connect the two users

### Location of key scripts from technical requriements
1. Delegates: Assets\Scripts\ManagersSingletons\GameManager
2. Singleton: Assets\Scripts\ManagersSingletons\GameManager
3. Audio: Assets\Scripts\ManagersSingletons\AudioManager
